#Четвертая проектная работа Место

##В данной работе закрепляются уже полученные в прошлых спринтах знания и навыки: флекс вёрстка, гриды, позиционирование элементов, применение трансформации к элементам. Также впервые отрабатываются полученные в последнем спринте знания и навыки java-script. Свёерстан и запрограммирован попап с работающими полями для ввода данных. Итак, это первая моя проектая работа, написанная на трех языка: HTML, CSS и JavaScript. Сайт свёрстан с применением адаптациии и отзывчивости к разным форматам экранов. Язык наименования блоков и элементов - БЭМ, файлы структурированы по методологии БЭМ Nested.
